# Studio Booking Platform

## Project Status

⚠️ Work in Progress

This project is currently under development and approximately 90% completed.

## Description

Studio Booking is a full-stack web application designed to simplify and modernize the process of reserving professional music recording studios through a centralized digital platform.

The system aims to provide artists, producers and studio administrators with an efficient environment for managing studio sessions, equipment availability and reservation workflows. The application follows a modern Client-Server architecture using React, Node.js, PostgreSQL and Prisma ORM, while authentication and authorization are handled through JWT and Google OAuth integration.

The main goal of the project is to eliminate scheduling conflicts, improve equipment visibility and provide a secure and user-friendly booking experience for both clients and administrators.

---

## Intended Features

- User registration and authentication
- Google OAuth login
- JWT-based authorization
- Studio room browsing
- Studio reservation management
- Equipment inventory management
- Equipment selection during booking
- Dynamic booking price calculation
- Booking history tracking
- Review and rating system
- Administrative dashboard
- Reservation approval and completion workflow
- Role-based access control (Artist / Administrator)
- Responsive user interface
- Modern Glassmorphism-inspired design
